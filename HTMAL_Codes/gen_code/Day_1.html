HTML (Hyper-text Markup Language):

o HTML is a language for describing web pages.
o HTML is not a programming language, it is a markup language.
o A markup language is a set of markup tags.
o HTML uses markup tags to describe web pages.
o HTML is W3C Recommendation.
o HTML language isn’t case- sensitive, and doesn’t affect by spaces and line breaks.
o HTML is Browser Dependant language.
